*vimai.txt*  Inline LLM queries and agents from Vim

vimai                                                          *vimai*

vimai sends prompts from Vim to an Azure OpenAI-compatible deployment,
shows responses in a scratch split, and supports prompt-only or external
agents through the same :AI command.

==============================================================================
INSTALLATION                                                   *vimai-install*

With vim-plug:
>
  Plug 'liupeirong/vimai', { 'do': 'uv sync' }
<

With Vundle:
>
  Plugin 'liupeirong/vimai'
<
After :PluginInstall, run `uv sync` in the plugin checkout.

With lazy.nvim:
>
  {
    "liupeirong/vimai",
    build = "uv sync",
  }
<

vimai uses the plugin checkout's `.venv` automatically when it exists. To use a
different Python, set one of these before the plugin loads:
>
  let g:vimai_python = '/path/to/python'
  let $VIMAI_PYTHON = '/path/to/python'
<

==============================================================================
COMMANDS                                                       *vimai-commands*

                                                        *:AI*
:AI {prompt}
        Send {prompt} to the LLM and show the response in [AI Response].

:AI @{agent} {prompt}
        Send a stateless single-turn prompt to a prompt-only or external agent.

                                                        *:AIPrompt*
:AIPrompt
        Open a scratch prompt buffer. Press <leader>s in that buffer to submit.

                                                        *:AISession*
:AISession
        Print the current vimai session file path.

Slash commands:
>
  :AI /clear   Close the current session and clear the response buffer.
  :AI /purge   Delete all vimai session files from the system temp directory.
  :AI /help    Show available vimai commands.
<

==============================================================================
CONFIGURATION                                                  *vimai-config*

Required environment variables:
>
  AZURE_OPENAI_ENDPOINT
  AZURE_OPENAI_DEPLOYMENT
<

Optional environment variables:
>
  LANGSMITH_API_KEY
  LANGSMITH_PROJECT
  VIMAI_EXTERNAL_AGENTS_DIR
  VIMAI_PYTHON
  VIMAI_SCRIPT
<

Run `:help vimai-install` for installation details and see README.md for full
setup, Azure authentication, and troubleshooting guidance.
